# 限俗小助手 - 超星学习通浏览器插件

## 插件简介

限俗小助手是一款专为超星学习通设计的浏览器插件，主要提供以下功能：

1. **解决英文内容不按空格换行问题**：自动为 `<span class="html-content-box">` 元素添加 `word-break:normal` 样式，确保英文内容能够正常换行。

2. **添加克隆活动按钮**：在学习通的活动列表页面，自动在"权限设置"按钮左侧添加"克隆活动"按钮，方便快速复制活动。

3. **展开/收起所有活动列表**：在活动列表页面添加切换按钮，可以一键展开或收起所有活动列表。

4. **作业中图片上方增加旋转按钮**：在作业中图片上方添加旋转按钮，点击后可以旋转图片，方便查看不同角度。

5. **一键导出所有签到活动数据**：在活动列表页面，自动在"克隆活动"按钮左侧添加"导出签到"按钮。点击该按钮，可以自动收集页面中所有签到活动的activeId，并按顺序在新标签页中打开导出页面，方便教师批量导出签到数据。

6. **作业列表添加"最大分页"按钮**：在作业列表页面，在"作业库"按钮后添加"最大分页"按钮。点击该按钮会自动选择每页显示50份，方便一次性查看更多作业。

7. **作业列表添加"隐藏零待批"按钮**：在作业列表页面，在"最大分页"按钮后添加"隐藏零待批"/"显示零待批"切换按钮。点击该按钮可以隐藏/显示待批数量为0的作业。

8. *作业列表添加"补交无期限"按钮**：在作业列表页面，在"最大分页"按钮后添加"补交无期限"按钮。点击该按钮对当前页面展示的作业逐个处理：如果没有打开补交，就打开补交，如果补交已打开，但是有期限，去掉期限。

## 支持的浏览器

- Google Chrome
- Microsoft Edge
- Mozilla Firefox

## 安装步骤

### 方法一：从 GitHub 下载 ZIP 文件

1. **下载插件源码**
   - 访问插件的 GitHub 仓库页面
   - 点击绿色的 "Code" 按钮
   - 选择 "Download ZIP" 选项
   - 等待 ZIP 文件下载完成

2. **解压文件**
   - 找到下载的 ZIP 文件（通常在下载文件夹中）
   - 右键点击文件，选择"解压"或"全部解压缩"
   - 确保解压后的文件夹名称为 `raw_study`

3. **在浏览器中加载插件**

   #### Chrome / Edge 浏览器
   1. 打开浏览器，输入 `chrome://extensions/`（Chrome）或 `edge://extensions/`（Edge）
   2. 在页面右上角找到并启用"开发者模式" (Edge在左侧三条横线点开后的下方启用"开发者模式")
   3. 点击"加载已解压的扩展程序"
   4. 选择解压后的 `raw_study` 文件夹
   5. 插件将自动安装，浏览器工具栏会显示插件图标

   #### Firefox 浏览器
   1. 打开 Firefox，输入 `about:debugging`
   2. 点击"此 Firefox"
   3. 点击"临时载入附加组件"
   4. 选择解压后的 `raw_study` 文件夹中的 `manifest.json` 文件
   5. 插件将临时安装，可以在浏览器工具栏看到插件图标

## 使用方法

1. 安装插件后，访问超星学习通网站，例如 https://mobilelearn.chaoxing.com/

2. 插件会自动生效：
   - 页面中的英文内容将能够正常换行
   - 活动列表会显示"克隆活动"和"导出签到"按钮
   - 在活动列表页面会显示"收回全部"/"展开全部"切换按钮
   - 在作业图片上方会显示旋转按钮
   - 在作业列表页面会显示"展示全部"和"未批改"按钮

3. **一键导出签到活动数据**：
   - 进入课程活动列表页面
   - 点击"导出签到"按钮
   - 插件会自动收集页面中所有签到活动的导出链接
   - 按顺序在新标签页中打开导出页面
   - 每个导出页面间隔500ms打开，避免被浏览器拦截

4. **作业列表快捷操作**：
    - 进入作业列表页面（如 `/mooc2-ans/mycourse/tch`）
    - 点击"最大分页"按钮可快速切换到每页显示50份
    - 点击"隐藏零待批"按钮可隐藏待批为0的作业，点击"显示零待批"恢复显示
    - 点击"批量增加补交时间"按钮可批量为所有作业增加补交截止时间
    - 点击"批量取消补交时间"按钮可批量取消所有作业的补交截止时间
    - 在"作答時間"后面会自动显示补交截止时间

5. **批阅页面快速打分**：
    - 进入作业批阅页面（如 `/epub-h5/?objectIds=...`）
    - 在页面底部的快速打分区域会显示"0分"、"60分"、"满分"三个按钮
    - 点击按钮可快速填入对应分数，60分按钮为蓝底白字，满分按钮为绿底白字

6. 点击插件图标可以查看插件信息和使用说明

## 常见问题

### Q: 如何更新插件？
A: 从 GitHub 重新下载最新版本，删除旧版本后重新加载新版本。

### Q: 点击"导出签到"按钮没有反应怎么办？
A: 请确保在课程活动列表页面使用，并且页面中确实存在签到活动。

### Q: 批量取消补交时间功能不工作怎么办？
A: 该功能需要浏览器允许弹窗。如果弹窗被拦截，请点击地址栏右侧的弹窗拦截图标，选择"总是允许此网站显示弹窗"，然后重新点击按钮。
